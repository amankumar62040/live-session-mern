# Client (Vite + React)

## Setup
1. Install dependencies:
   ```
   cd client
   npm install
   ```
2. Run dev server:
   ```
   npm run dev
   ```
3. Open http://localhost:5173
4. Click START SESSION to create an admin session and see the admin video player.
5. Copy the Student URL to open the same video player as the student.

The client uses a public sample video by default. You can change VITE_SAMPLE_VIDEO in `.env`.
